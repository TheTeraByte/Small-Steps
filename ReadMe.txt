Project Installation Instructions: 

Small Steps is a habit tracking Android app designed for Android devices version 1.6 and higher. For optimum use run the application on a medium screen sized Android phone with external storage. The presence of external storage (in the form of microSD card) is essential. 

I have provided an installable .apk file within my submission folder.
